\# Architecture - Two Agent System



\### Agent Setup \& Routing

\- \*\*Brain (Orchestrator):\*\* Hermes Agent powered by Gemini 2.5-Flash (Free Tier) / Groq for high-level planning.

\- \*\*Hands (Execution):\*\* OpenClaw powered by Ollama (qwen2.5-coder) running locally to execute commands and write files.



\### Slack Channel Scheme

\- `#sprint-main`: For high-level goals and approval.

\- `#agent-coder`: For OpenClaw task execution logs.

\- `#agent-log`: For raw autonomous activities and crons.
